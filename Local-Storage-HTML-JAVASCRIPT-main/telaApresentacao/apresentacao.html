Este site foi criado com o intuito de teste para o curso de Desenvolvimento de Sistemas realizado na instituição SENAI de Jaraguá do Sul.
Ele foi construído com a junção da linguagem HTML, estilização do site em CSS, JavaScript para a parte de lógica e o uso do Local Storage para armazenamento dos dados cadastrados.
O site conta com as funções de cadastrar um usuário e consultar os usuários já cadastrados juntamente com botões para navegar entre as páginas.
